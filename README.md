###Memory game works on macOS, Windows, and Linux.
Для запуска игры необходимо установить все зависимости:

    npm install

Перед запуском приложения, необходимо запустить сервер: 
    
    npm run server

После запуска сервера, запускается приложение:
    
    npm run start
